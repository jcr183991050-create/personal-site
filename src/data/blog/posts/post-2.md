# Tailwind CSS入门指南

Tailwind CSS是一个实用的CSS框架，可以帮助您快速构建自定义设计。

## 优点

- **实用类优先**: 直接在HTML中使用类名
- **响应式设计**: 内置响应式工具
- **定制化**: 完全可配置的设计系统

## 基本使用

```html
<button class="px-4 py-2 bg-blue-600 text-white rounded hover:bg-blue-700">
  点击我
</button>
```

## 响应式示例

```html
<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4">
  <!-- 内容 -->
</div>
```
